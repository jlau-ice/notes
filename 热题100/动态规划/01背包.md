```java
public static int solve(int N, int C, int[] v, int[] w) {
    int[][] dp = new int[N + 1][C + 1];
    // dp[i][j] 表示 前 i 个物品，在背包容量为 j 时的最大价值
    for (int i = 1; i <= N; i++) {
        for (int j = 1; j <= C; j++) {
            if (j >= v[i - 1]) {
                dp[i][j] = Math.max(dp[i - 1][j - v[i - 1]] + w[i - 1], dp[i - 1][j]);
            } else {
                dp[i][j] = dp[i - 1][j];
            }
        }
    }
    return dp[N][C];
}
```

空间复杂度优化 压缩为一维数组。

```java
public static int solve(int N, int C, int[] v, int[] w) {
    int[] dp = new int[C + 1];
    for (int i = 1; i <= N; i++) {
        for (int j = C; j > 0; j--) {
            if (j >= v[i - 1]) {
                // 这里dp[j] 就是上一轮的，还没有被滚动更新
                dp[j] = Math.max(dp[j - v[i - 1]] + w[i - 1], dp[j]);
            } 
        }
    }
    return dp[C];
}
```
