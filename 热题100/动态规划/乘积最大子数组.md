
[题目地址](https://leetcode.cn/problems/maximum-product-subarray/)

需要注意最大值和最小值 有三种可能 一个是和上一个最大值 乘当前值   当前值  上一个最小值乘当前位置，就是要考虑负负得正的情况。

```java
class Solution {
    public int maxProduct(int[] nums) {
        long[] dpMax = new long[nums.length];
        long[] dpMin = new long[nums.length];
        dpMax[0] = nums[0];
        dpMin[0] = nums[0];
        long max = nums[0];
        for (int i = 1; i < nums.length; i++) {
            dpMin[i] = Math.min(Math.min(dpMin[i - 1] * nums[i], nums[i]), dpMax[i - 1] * nums[i]);
            dpMax[i] = Math.max(Math.max(dpMin[i - 1] * nums[i], nums[i]), dpMax[i - 1] * nums[i]);
            max = Math.max(dpMax[i], max);
        }
        return (int) max;
    }
}
```
