
[题目地址](https://leetcode.cn/problems/integer-break/)

我一开始写的 是 `dp[i] = Math.max(dp[i], (j * dp[i - j]));`
调试发现不对，后面可以选择拆分或者不拆分。
正确应该是 `dp[i] = Math.max(dp[i], Math.max(j * (i - j), (j * dp[i - j])));`
我一开始的写法相当于不要后面的 。到`dp[4]` 就错了。

```
dp[1] = 1

dp[2]
1 * dp[1] || 1 * 1

dp[3]
1 * dp[2] || 1 * 2
2 * dp[1] || 2 * 1

dp[4]
1 * dp[3] || 1 * 3
2 * dp[2] || 2 * 2
3 * dp[1] || 3 * 1
```

```java
class Solution {
    public int integerBreak(int n) {
        int[] dp = new int[n + 1];
        dp[0] = 0;
        dp[1] = 1;
        if (n < 2) {
            return n;
        }
        for (int i = 2; i <= n; i++) {
            for (int j = 1; j < i; j++) {
                dp[i] = Math.max(dp[i], Math.max(j * (i - j), (j * dp[i - j])));
            }
        }
        return dp[n];
    }
}
```
