[题目地址](https://leetcode.cn/problems/target-sum/)

和等和子集差不多，这里要找的是 和为 `(sum + target) / 2` 的。

```java
class Solution {
    public int findTargetSumWays(int[] nums, int target) {
        // 看了卡尔如何把问题转化为背包问题，为应该会写了
        // sum 确定 target 确定 只要找能凑出 sum - target 的就行了
        // 这就变成典型的背包问题了
        // 错了这是组合数问题 不是最大值问题
        int sum = 0;
        for (int i = 0; i < nums.length; i++) {
            sum += nums[i];
        }
        int p = sum + target;
        int q = sum - target;
        // 推断出 p 必为偶数 因为sum 和target 必须是同属性 。要么同基 要么同偶。
        // 必须是非负数。
        if (p % 2 != 0 || p < 0) {
            return 0;
        }
        int c = p / 2;
        // dp[j] 表示 方法数量？
        int[] dp = new int[c + 1];
        dp[0] = 1;
        for (int i = 1; i <= nums.length; i++) {
            for (int j = c; j >= nums[i - 1]; j--) {
                dp[j] += dp[j - nums[i - 1]];
            }
        }
        return dp[c];
    }
}
```
