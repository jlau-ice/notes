这题和昨天[路径总和II](../树/路径总和II.md)很像，看了一遍卡代码随想录，感觉很熟悉。
需要注意的点就是 startIndex 是 i + 1，不是startIndex + 1。被卡尔误导了。

```java
class Solution {
    List<List<Integer>> res = new ArrayList<>();
    List<Integer> list = new ArrayList<>();

    public List<List<Integer>> combine(int n, int k) {
        backtring(n, k, 1);
        return res;
    }

    public void backtring(int n, int k, int startIndex) {
        if (list.size() == k) {
            res.add(new ArrayList<>(list));
            return;
        }
        for (int i = startIndex; i <= n; i++) {
            list.add(i);
            backtring(n, k, i + 1);
            list.remove(list.size() - 1);
        }
    }
}
```
