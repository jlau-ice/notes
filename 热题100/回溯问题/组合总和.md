

这里需要注意的是可以重复使用一个数，那么就不需要startIndex,所有的遍历都是从0开始。

但是需要注意一点就是，显然需要从小往大的加，如果给的数组是乱序的，显然不行，需要提前个给数组拍个序

```java
class Solution {

    List<List<Integer>> res = new ArrayList<>();
    List<Integer> list = new ArrayList<>();

    public List<List<Integer>> combinationSum(int[] candidates, int target) {
        Arrays.sort(candidates);
        backtacking(candidates, target, 0, 0);
        return res;
    }

    public void backtacking(int[] candidates, int target, int sum, int startIndex) {
        if (sum == target) {
            res.add(new ArrayList<>(list));
            return;
        }
        // 这里为啥不是0,不是可以重复选数组吗？，这里只是然结果不要重复。
        // 可以重复选，不代表可以选前面的啊。
        // 选前面的不久重复了吗？
        for (int i = startIndex; i < candidates.length; i++) {
            if (sum + candidates[i] > target) {
                break;
            }
            list.add(candidates[i]);
            // 其实在这里说明了可以选相同的数字。
            backtacking(candidates, target, sum + candidates[i], i);
            list.remove(list.size() - 1);
        }

    }
}
```
