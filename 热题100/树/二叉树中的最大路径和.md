[题目地址](https://leetcode.cn/problems/binary-tree-maximum-path-sum)

这题居然这么简单，有的想二叉树最大直径。
二叉树的最大直径中就是计算左边最大深度，右边最大深度。
直径就是他们相加

这个有点类似。

如果为空返回0

如果有子树，返回 和左边 构建的大还是右边构建的大。

计算一下当前最大值。
当前最大值是 左边最大值 + 右边最大值 + 当前节点的值
注意的是左边最大值如果是负数，则不参与计算。

```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    Integer max = Integer.MIN_VALUE;

    public int maxPathSum(TreeNode root) {
        maxx(root);
        return max;
    }

    public int maxx(TreeNode root) {
        if (root == null) {
            return 0;
        }
        int l = Math.max(maxx(root.left), 0);
        int r = Math.max(maxx(root.right), 0);
        int curMax = l + r + root.val;
        max = Math.max(max, curMax);
        return root.val + Math.max(l, r);
    }
}
```
