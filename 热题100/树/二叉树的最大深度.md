
你应该无条件的相信你的递归函数，他会给你正确的结果。
你不用去管递归的实现过程，你只要知道，他会给你正确的结果就对了。其他的不用去管。

以这题为例，二叉树的最大深度就是，左孩子的最大深度和右孩子最大深度之和加1。
我要获取左孩子最大深度，我就调用这个函数。右孩子最大深度就调函数传入右孩子。


```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public int maxDepth(TreeNode root) {
        if (root == null) {
            return 0;
        }
        int l = maxDepth(root.left);
        int r = maxDepth(root.right);
        return Math.max(l, r) + 1;
    }

}
```
