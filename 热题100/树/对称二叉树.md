[题目地址](https://leetcode.cn/problems/symmetric-tree)

思路就是 定义两个指针指向 左右子树，如何都为空 返回false
如果有一个为空返回false
如果值不同 返回false
如果值相等
则继续比较他的做孩子 也另外一个的右孩子。 他的右孩子 也和另一个的左孩子比较。
递归实现
```java
/**
 * Definition for a binary tree node.
 * public class TreeNode {
 *     int val;
 *     TreeNode left;
 *     TreeNode right;
 *     TreeNode() {}
 *     TreeNode(int val) { this.val = val; }
 *     TreeNode(int val, TreeNode left, TreeNode right) {
 *         this.val = val;
 *         this.left = left;
 *         this.right = right;
 *     }
 * }
 */
class Solution {
    public boolean isSymmetric(TreeNode root) {
        return sy(root.left, root.right);
    }
    public boolean sy(TreeNode n1, TreeNode n2) {
        if (n1 == null && n2 == null) {
            return true;
        }
        if (n1 == null || n2 == null) {
            return false;
        }
        // 不能写成 值相等返回true, 要继续比骄傲子树，直到节点都为空。
        if (n1.val != n2.val) {
            return false;
        }
        return sy(n1.left, n2.right) && sy(n1.right, n2.left);
    }
}
```
