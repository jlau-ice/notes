I和II不同就在于 I不是有序的。二世有序的 这个很重要, 但是所有测试用例重复的数字都会连着，不会出现断断续续的情况。

I的解法：

```java
class Solution {
    public int removeDuplicates(int[] nums) {
        int l = 0;
        for (int r = 0; r < nums.length; r++) {
            if (nums[l] != nums[r]) {
                nums[l + 1] = nums[r];
                l = l + 1;
            }
        }
        return l + 1;
    }
}
```

II的解法：
```java
class Solution {
    public int removeDuplicates(int[] nums) {
        int l = 2;
        for (int r = 2; r < nums.length; r++) {
            if (nums[l - 2] != nums[r]) {
                nums[l] = nums[r];
                l++;
            }
        }
        return l;
    }
}
```
