解法1：贪心 记录前面的最小值 ，计算每个每个位置所能卖出的最大值。

```java
class Solution {
    public int maxProfit(int[] prices) {
        int res = 0;
        int min = prices[0];
        for (int i = 1; i < prices.length; i++) {
            if (min > prices[i]) {
                min = prices[i];
            }
            res = Math.max(res, prices[i] - min);
        }
        return res;
    }
}
```

动态规划的写法
```java
class Solution {
    public int maxProfit(int[] prices) {
        // 动态规划写法
        int n = prices.length;
        int[][] dp = new int[n][2];
        dp[0][0] = -prices[0];
        // dp[0][1] = 0;
        for (int i = 1; i < n; i++) {
            dp[i][0] = Math.max(dp[i - 1][0], -prices[i]);
            dp[i][1] = Math.max(dp[i - 1][0] + prices[i], dp[i - 1][1]);
        }
        return dp[n - 1][1];
    }
}
```
