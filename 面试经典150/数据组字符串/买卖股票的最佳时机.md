[题目地址](https://leetcode.cn/problems/best-time-to-buy-and-sell-stock)

解法1：贪心 记录前面的最小值 ，计算每个每个位置所能卖出的最大值。

```java
class Solution {
    public int maxProfit(int[] prices) {
        int res = 0;
        int min = prices[0];
        for (int i = 1; i < prices.length; i++) {
            if (min > prices[i]) {
                min = prices[i];
            }
            res = Math.max(res, prices[i] - min);
        }
        return res;
    }
}
```
