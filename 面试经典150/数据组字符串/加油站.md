[加油站](https://leetcode.cn/problems/gas-station/)

如果从 A 出发，最远只能走到 B（无法到达 B 的下一站），那么 A 到 B 之间的任何一个城市出发，都无法到达 B 的下一站。

这是这道题最神奇的地方。因为：
我们已经通过 totalSum >= 0 确定了全城有足够的油。
我们通过循环排除了前面所有不合格的起点，剩下的这个 start 是唯一能跑完剩余路程的。
既然总油量是够的，而 start 之后的路程又是合法的，那么把剩下的油带回到开头跑完剩下的半圈也一定够。

```java
class Solution {
    public int canCompleteCircuit(int[] gas, int[] cost) {
        int start = 0;
        int total = 0;
        int cur = 0;
        for (int i = 0; i < gas.length; i++) {
            total += gas[i] - cost[i];
            cur += gas[i] - cost[i];
            if (cur < 0) {
                cur = 0;
                start = i + 1;
            }
        }
        if (total >= 0) {
            return start;
        } else {
            return -1;
        }
    }
}
```
