[题目地址](https://leetcode.cn/problems/merge-sorted-array)

用合并两个有序链表的思路就行了。但是需要从后面便利，大的往 nums1 后面放就行了 
如果 nums2 先放完了 直接返回就行了 
如果 nums1 先放完了 那就依次把 nums2 补到前面就行了

```java
class Solution {
    public void merge(int[] nums1, int m, int[] nums2, int n) {
        int cur1 = m - 1;
        int cur2 = n - 1;
        int cur = m + n - 1;
        while (cur1 >= 0 && cur2 >= 0) {
            if (nums1[cur1] > nums2[cur2]) {
                nums1[cur] = nums1[cur1];
                cur1--;
            } else {
                nums1[cur] = nums2[cur2];
                cur2--;
            }
            cur--;
        }
        if (cur1 < 0) {
            while (cur >= 0) {
                nums1[cur] = nums2[cur];
                cur--;
            }
        }
    }
}
```
