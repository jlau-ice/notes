投票 count = 0 一下个就是候选人 相同 ++ 不同 -- ，需呀注意就是设定候选人时候就要++;
```java
class Solution {
    public int majorityElement(int[] nums) {
        // 空间O(n)很简单，怎么才能空间 O(1) 呢？
        int res = -1;
        int count = 0;
        for (int i = 0; i < nums.length; i++) {
            if (count == 0) {
                res = nums[i];
            }
            if (res == nums[i]) {
                count++;
            } else {
                count--;
            }
        }
        return res;
    }
}
```
