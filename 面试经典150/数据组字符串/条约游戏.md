考虑是否可以覆盖整个数组长度就行了，做适当的剪枝

```java
class Solution {
    public boolean canJump(int[] nums) {
        int len = 0;
        for (int i = 0; i < nums.length; i++) {
            if (len < i) {
                return false;
            }
            len = Math.max(nums[i] + i, len);
            if (len >= nums.length - 1) {
                return true;
            }
        }
        return false;
    }
}
```
