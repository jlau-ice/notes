写法1：左右指针分别指向两头，如果左边指向的值等于目标值，那么就左边的值就等于右指针的值（不是交换）。然后右边指针往前移动一位 
如果左边指针不等于目标值 则左指针往后移。最后返回的结果就是左边指针索引值。

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int l = 0;
        int r = nums.length - 1;
        while (l <= r) {
            if (nums[l] == val) {
                nums[l] = nums[r];
                r--;
            } else {
                l++;
            }
        }
        return l;
    }
}
```


一个是分别都指向开始，如果右边指针不等于目标值，那么右边指针的值赋值给左指针指向的位置，两个指针都往后移动一个单位。
如果右边指针等于目标值，就往后移动。另外一个原地等着，直到右指针不等于目标值。就执行上面那个逻辑。

```java
class Solution {
    public int removeElement(int[] nums, int val) {
        int n = nums.length;
        int left = 0;
        for (int right = 0; right < n; right++) {
            if (nums[right] != val) {
                nums[left] = nums[right];
                left++;
            }
        }
        return left;
    }
}
```
